# Support

## Getting Help

1. Check project docs:
   - `README.md`
   - `docs/README.md`
2. Review troubleshooting and runtime logs
3. Open an issue with:
   - What you expected
   - What happened
   - Steps to reproduce
   - Environment details (OS, Node, pnpm, Docker)

## Runtime Checks

Useful commands:

```bash
pnpm docker:ps
pnpm docker:logs
pnpm build
pnpm typecheck
```
